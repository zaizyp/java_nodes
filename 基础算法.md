## 基础算法
### 排序类
- 冒泡排序：
```java
public void bubbleSort(int[] arr){
    for(int i = 0; i < arr.length-1; i++) {
        for(int j = 0; j < arr.length-1; j++) {
            if(arr[j] > arr[j+1]) {
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}
```
- 选择排序：
```java
public void selectSort(int[] arr){
    for(int i = 0; i < arr.length-1; i++){
        for(int j = i+1; j < arr.length-1; j++){
            if(arr[i] > arr[j]){
                int temp = arr[i];
                arr[i] = arr[j];
                arr[j] = temp;
            }
        }
    }
}
```
